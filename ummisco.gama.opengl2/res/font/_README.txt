The font files here are generated by the software "hiero".
To generate other fonts, please follow the instructions on this video, from 1:30 to 3:00 :
https://www.youtube.com/watch?v=d8cfgcJR9Tk&index=33&list=PLRIWtICgwaX0u7Rf9zkZhLoLuZVfUksDP

After following those instructions (please generate the font to "ASCII" and not in "NEHE" if you want to have
special characters), edit your ".fnt" file generated, removing all the lines starting with "kerning".
The files must be named with the following convention : 
- Fontname
- Fontname bold
- Fontname italic
- Fontname bold italic
